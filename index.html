NEWSIGHT

비지도 분류 학습 모델을 이용한 뉴스 기사 데이터 분류 서비스입니다. 뉴스기사를 이용하여 많은 정보들을 필요로하는 사람들에게 기존 뉴스 표시 방식보다 빠르고 유의미한 뉴스 데이터를 제공해주는 서비스입니다. 사용 언어는 python, 개발도구 pycharm, jupyter notebook 이며 프레임웍은 django framework 을 사용하였습니다. 크롤링 및 bigkinds api를 이용하여 약 16만건의 뉴스 데이터를 가져왔으며 전처리를 통해 유효한 뉴스 기사를 먼저 분류 , top-n 과 optics 알고리즘을 이용하여 분류 모델을 사용하였습니다. 이것을 바탕으로 Elasticsearch 의 검색 알고리즘 bm25를 이용해 검색을 하였습니다.

TFOX(ERP)

응용 어플리케이션의 대표적인 ERP를 직접 구현함으로써 개발 역량을 키우는 목표로 진행하였습니다. 담당 역할은 형상관리를 맡았으며 git을 이용한 버전관리를 하였습니다. 주된 구현기능은 전자결재를 맡았으며 기본적인 전자결재 프로세스를 구현하였고 signiturePad 를 이용한 사용자 개인별 사인 관리 기능을 구현, toast ui editor를 이용한 양식 관리 기능, 양식 작성 기능을 구현하였습니다. 사용 언어는 java, 개발도구는 sts, sql developer , da#5 이며 프레임웍은 Spring Framework 를 이용하였습니다. maven 을 이용하여 기본 설정들을 관리하였으며 mybatis를 이용하여 sql문을 따로 관리하였습니다. 서버는 tomcat을 활용하였습니다.
